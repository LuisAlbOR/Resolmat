Librerías necesarias
PLY (Python Lex-Yacc):

Es la librería principal que utilizas para implementar el lexer y el parser.

Proporciona las herramientas lex y yacc para el análisis léxico y sintáctico.

Tkinter:

Es la librería estándar de Python para crear interfaces gráficas.

No necesitas instalarla, ya que viene incluida con Python.

ttk (Themed Tkinter):

Es una extensión de Tkinter que proporciona widgets con un aspecto más moderno.

Tampoco necesitas instalarla, ya que viene incluida con Tkinter.

instalar todas las dependencias ejecutando:
    pip install -r requirements.txt
